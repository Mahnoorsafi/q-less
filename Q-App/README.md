# QMe-App

This workspace contains two frontend apps:

- `QMe-Customer`: React Native mobile app built with Expo
- `QMe-Admin`: React web admin panel built with Create React App structure
- `firebase-functions`: placeholder for Firebase Cloud Functions

## Setup

### Customer app

1. Open `QMe-Customer`
2. Run `npm install`
3. Run `npm start`

### Admin app

1. Open `QMe-Admin`
2. Run `npm install`
3. Run `npm start`

## Notes

- Firebase config files are placeholders. Replace `YOUR_API_KEY`, `YOUR_PROJECT_ID`, etc. with actual Firebase settings.
- The customer app uses mock data and navigation screens for preview.
- The admin panel contains pages for login, dashboard, queue monitoring, services, analytics, and notifications.
